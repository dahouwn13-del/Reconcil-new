# CIEL RECONCILIATION — Version 5.2.0

Booking.com reconciliation for Opera PMS with a separate live Work Log window.

## Version 5.2 additions

- Local smart name matching with no cloud API and no guest data leaving the computer.
- Arabic/Persian and Cyrillic transliteration.
- Known international spelling variations, fuzzy token comparison, initials and phonetic comparison.
- Smart name confidence score and explanation in the results grid and Excel export.
- Arrival +1 day with matching guest and departure remains **Perfect Match**.
- Such records show **Action Required: Check if No-Show was charged**.
- Added Action Required and Smart Name Analysis columns.

The Booking.com reader, Opera PDF reader, split-stay detection, dashboard, Excel export and separate Work Log remain available.
